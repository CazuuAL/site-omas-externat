# Guide Enseignant - OMAS Externat

## 🎓 Bienvenue sur OMAS Externat

Ce guide vous explique comment créer et gérer des QCM pour vos étudiants.

## 📝 Inscription en tant qu'Enseignant

1. Accédez à la page de connexion
2. Cliquez sur l'onglet "Inscription"
3. **Important** : Sélectionnez "Enseignant" dans le champ "Je suis :"
4. Remplissez vos informations (nom, prénom, email, mot de passe)
5. Cliquez sur "S'inscrire"

## 🔑 Connexion

1. Utilisez votre email et mot de passe
2. Vous serez automatiquement redirigé vers votre dashboard enseignant

## 📊 Votre Dashboard

Votre dashboard affiche :
- La liste de tous vos QCM créés
- Le statut de chaque QCM (publié/brouillon)
- Des actions rapides (modifier, publier, supprimer)

### Actions disponibles :
- 👁️ **Publier** : Rend le QCM visible aux étudiants
- 👁️‍🗨️ **Dépublier** : Cache le QCM aux étudiants
- ✏️ **Modifier** : Éditer le contenu du QCM
- 🗑️ **Supprimer** : Supprimer définitivement le QCM

## ✨ Créer un Nouveau QCM

### Étape 1 : Informations Générales

Cliquez sur "Créer un nouveau QCM" et remplissez :

1. **Titre** : Nom descriptif du QCM
   - Exemple : "Cardiologie - Insuffisance cardiaque"

2. **Spécialité** : Choisir dans la liste
   - Cardiologie, Pneumologie, Neurologie, etc.

3. **Semaine** : Numéro de semaine (1-52)

4. **Description** : Résumé du contenu (optionnel)

5. **Dates** :
   - **Disponible à partir du** : Date de début
   - **Disponible jusqu'au** : Date de fin
   - **Date limite** : Deadline pour les corrections

### Étape 2 : Ajouter des Questions

Pour chaque question :

1. **Énoncé** : La question posée aux étudiants
   - Soyez précis et clair
   - Exemple : "Quel est le traitement de première intention de l'IC à FE réduite ?"

2. **Options de réponse** :
   - **Réponses A, B, C, D** : Obligatoires
   - **Réponse E** : Optionnelle
   - Variez l'ordre des réponses pour éviter les patterns

3. **Réponse correcte** : Sélectionnez A, B, C, D ou E

4. **Explication** : Justification pédagogique
   - Expliquez POURQUOI cette réponse est correcte
   - Ajoutez des références si possible
   - Exemple : "Les IEC (ou ARA2) sont le traitement de première ligne avec preuves de réduction de mortalité"

### Étape 3 : Ajouter Plus de Questions

- Cliquez sur "➕ Ajouter une question" autant de fois que nécessaire
- Vous pouvez supprimer une question avec le bouton ❌ en haut à droite
- Pas de limite au nombre de questions

### Étape 4 : Sauvegarder

- **Annuler** : Retour au dashboard sans sauvegarder
- **💾 Créer le QCM** : Sauvegarder (le QCM sera en brouillon)

## 📤 Publier un QCM

Un QCM créé est automatiquement en **brouillon** :
- ❌ Les étudiants ne le voient PAS
- ✅ Vous pouvez le modifier librement

Pour le publier :
1. Allez sur votre dashboard
2. Trouvez le QCM
3. Cliquez sur l'icône 👁️
4. Confirmez la publication

Une fois publié :
- ✅ Les étudiants peuvent y accéder
- ✅ Visible dans la liste des QCM hebdomadaires
- ⚠️ Vous pouvez toujours le modifier ou le dépublier

## 🔄 Modifier un QCM

1. Cliquez sur l'icône ✏️ dans votre dashboard
2. Modifiez les informations ou questions
3. Sauvegardez les modifications

**Note** : Les modifications sont instantanées, même pour un QCM publié.

## 🗑️ Supprimer un QCM

1. Cliquez sur l'icône 🗑️
2. Confirmez la suppression

**⚠️ Attention** : Cette action est **irréversible** !

## 💡 Bonnes Pratiques

### Pour les Questions

✅ **À FAIRE** :
- Utilisez un langage médical précis
- Variez les niveaux de difficulté
- Basez-vous sur les recommandations actuelles
- Donnez des explications détaillées
- Relisez pour éviter les ambiguïtés

❌ **À ÉVITER** :
- Questions trop faciles ou évidentes
- Formulations ambiguës
- Réponses trop similaires
- Pièges ou questions "tricks"

### Pour l'Organisation

✅ **À FAIRE** :
- Créez des QCM réguliers (chaque semaine)
- Respectez les dates limites
- Variez les spécialités
- Testez vos QCM avant de publier
- Dépubliez après la date limite

❌ **À ÉVITER** :
- Publier trop de QCM d'un coup
- Modifier un QCM après que les étudiants ont répondu
- Dates incohérentes (fin avant début)

### Pour le Timing

**Recommandation** :
- Créez le QCM en avance (quelques jours avant)
- Laissez-le en brouillon pour relecture
- Publiez au moment opportun
- Dépubliez après la date limite pour révéler les corrections

## 🎯 Exemple Complet

Voici un exemple de QCM bien construit :

**Titre** : Cardiologie - Insuffisance Cardiaque  
**Spécialité** : Cardiologie  
**Semaine** : 15  
**Description** : QCM sur la prise en charge de l'insuffisance cardiaque à fraction d'éjection réduite

**Question 1** :  
*Énoncé* : Quel est le traitement de première intention de l'IC à FE réduite ?  
- A. Inhibiteur calcique
- B. IEC ou ARA2 ✅
- C. Digoxine
- D. Diurétique seul

*Explication* : Les IEC (ou ARA2 en cas d'intolérance) sont le traitement de première ligne de l'IC à FE réduite, avec preuves de réduction de mortalité selon les recommandations ESC 2021.
